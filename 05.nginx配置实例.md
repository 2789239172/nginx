## 反向代理 

### (一) 打开浏览器, 在浏览器地址栏输入地址 跳转百度主页面

1. 具体实现

   - 对外开放端口

     firewall-cmd --add-port=8080/tcp --permanent

     firewall-cmd -reload

   - 查看已经开放的端口

     firewall-cmd --list-all

   - 添加代理

     ```nginx
     http {
       server{
         location / {
      	  proxy_pass https://www.baidu.com/;
         }
       }
     }
     ```
